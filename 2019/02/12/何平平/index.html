Android开发工程师 微企信息

项目起始时间:   2018/7-至今

项目描述

中铁出行是一个网约车平台，主要包括后台管理系统+乘客APP端+司机APP端组成。

  1. 后台管理中主要内容是乘客管理、司机管理、财务管理、调度中心等；
  2. 乘客APP主要内容是寻车下单->司机接单->行程结束->乘客支付这套流程，以及乘客个人信息管理等；
  3. 司机端APP主要内容是抢单->导航定位计价->完成订单->收款这套流程，以及司机提现和司机信息管理等。

工作内容

  1. 负责中铁出行乘客端的功能开发和维护;
  2. 集成实人认证，语音识别SDK；
  3. 通过高德地图实现导航和路线规划；
  4. APP用户数据缓存优化；
  5. 封装Retrofit+RxJava+OkHttp网络请求框架。

Android 开发实习生 年享资产

项目起始时间:   2018/3-2018/6

项目描述

IDAX交易所项目是一个数字货币交易平台，用户通过该平台来进行币币交易。分为四个模块：

  1. 市场，显示交易对的基本信息，并用k线显示该交易对的市场交易情况；
  2. 交易，该模块显示最新的10条买价和卖价，用户可在此模块进行数字币的买入卖出；
  3. 资产，显示当前帐户的预估总资产和各币种，并可进行充值和提现；
  4. 用户中心，可对用户的各种信息进行查看和设置。

工作内容

在「IDAX交易所项目组」负责IDAX Android端的部分开发工作，主要内容如下：

  1. 根据UI给的原型，绘制APP的界面；
  2. 根据产品需求，完成功能流程；
  3. 根据产品定位，进行兼容适配、屏幕适配和语言的适配；
  4. 在APP中使用第三方sdk进行埋点和安全验证；
  5. 使用第三方库MPChart进行K线的绘制；
  6. 对APP进行内存优化，提高流畅性和稳定性。
